# Genuary 2026

Projects, sketches, and experiments following prompts from [Genuary 2026](https://genuary.art/).

Using
[p5.js](https://p5js.org/)
with
[TypeScript](https://www.typescriptlang.org/)
and
[webpack](https://webpack.js.org/).
